# What's This

Library collection for Phalcon 4.

# Install

```console
$ composer require shucream0117/phalcon-libs
```

# Testing
```console
$ docker compose run --rm app vendor/bin/phpunit
```

# Composer update
```console
$ docker compose run --rm app composer update
```
